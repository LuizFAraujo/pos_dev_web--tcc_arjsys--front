# ArjSys - Sistema de Gerenciamento de Projetos Mecânicos

Frontend desenvolvido como TCC de Pós-graduação em Desenvolvimento Web.

## Tecnologias

- React 19
- TypeScript
- Vite 7
- Tailwind CSS 4.1
- TanStack Router
- Zustand

## Instalação

```bash
cd app
pnpm install
pnpm dev
```

## Estrutura

- **app/** - Aplicação frontend
- **docs/** - Documentação técnica
